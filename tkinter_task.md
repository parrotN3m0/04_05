### Задание 1. «Исчезающая кнопка» (простое)
Создайте окно с одной кнопкой. При нажатии на кнопку она должна **уничтожиться** (исчезнуть). Используйте `place` для размещения.

---

### Задание 2. «Привет, мир!» (Entry + Label)
Создайте окно, в котором есть:
- Поле для ввода (`Entry`)
- Кнопка с текстом «Сказать привет»
- Метка (`Label`) для вывода сообщения

При нажатии на кнопку программа должна взять текст из поля ввода и вывести его в метке в формате:  
**«Привет, [введённый текст]!»**


---

### Задание 3. «Светофор» (смена цвета через кнопку)
Создайте окно, на котором расположены:
- Метка размером 100×100 пикселей (например, с фоном `"grey"`)
- Три кнопки: «Красный», «Жёлтый», «Зелёный»

При нажатии на кнопку фон метки меняется на соответствующий цвет.  

---

### Задание 4. «Личный счётчик» (работа с текстом кнопки)
Создайте окно с:
- Меткой, на которой написан текущий счёт (например, «0»)
- Кнопкой «+1»
- Кнопкой «Сброс»

При нажатии на «+1» число в метке увеличивается на 1, при нажатии «Сброс» возвращается в 0.  
*Подсказка: чтобы изменить текст в метке, используйте `lbl["text"] = str(новое_значение)`.*

---

### Задание 5. «Калькулятор в 2 поля» (сетка grid)
Используя `grid`, создайте интерфейс:
- Два поля ввода (`Entry`) для чисел
- Четыре кнопки: «+», «-», «*», «/»
- Метка для вывода результата

При нажатии на кнопку выполняйте соответствующую операцию над числами из полей ввода и выводите результат в метку.

| Что используется | Где применяется | Краткое описание |
|----------------|----------------|------------------|
| `tkinter.Tk()` | создание главного окна | Возвращает объект главного окна (`root`). |
| `root.mainloop()` | запуск приложения | Запускает цикл обработки событий (делает окно интерактивным). |
| `root.title("текст")` | заголовок окна | Устанавливает текст в строке заголовка окна. |
| `root.geometry("ширинаxвысота")` | размер окна | Задаёт начальный размер окна, например `"300x200"`. |
| `tkinter.Button(родитель, text="...", command=функция)` | создание кнопки | Создаёт кнопку с подписью. При нажатии вызывается указанная функция. |
| `tkinter.Label(родитель, text="...", background="цвет")` | создание метки | Создаёт текстовую/цветную область на окне. |
| `tkinter.Entry(родитель)` | поле ввода | Создаёт однострочное поле для ввода текста. |
| `tkinter.Frame(родитель)` | контейнер | Невидимый прямоугольник, внутри которого можно группировать виджеты. |
| `виджет.place(x=..., y=..., width=..., height=...)` | абсолютное позиционирование | Размещает виджет в указанных пиксельных координатах и задаёт его размер. |
| `виджет.grid(row=..., column=..., padx=..., pady=...)` | позиционирование в сетке | Размещает виджет в ячейке воображаемой таблицы. `padx/pady` – отступы. |
| `виджет.destroy()` | удаление виджета | Полностью уничтожает виджет (убирает с экрана и освобождает память). |
| `виджет["текстовое_свойство"] = значение` | изменение свойств | Меняет внешний вид или поведение виджета (например, `lbl["text"] = "новый текст"`, `lbl["background"] = "red"`). |
| `виджет.cget("свойство")` | чтение свойства | Возвращает текущее значение свойства (например, `btn.cget("bg")` – цвет фона). |
| `виджет.config(свойство=значение)` | настройка виджета | Альтернативный способ изменения свойств (например, `btn.config(bg="white")`). |
| `entry.get()` | получить текст из поля ввода | Возвращает строку, введённую пользователем в `Entry`. |
| `root.after(мс, функция)` | таймер | Вызывает указанную функцию через заданное количество миллисекунд. |
| `str(число)` | преобразование в строку | Используется для вывода чисел в метках (т.к. `text` требует строку). |
