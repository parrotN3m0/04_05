
### Инкапсуляция (скрытие данных)

**Пример 1. Банковский счёт – защита баланса**
```python
class BankAccount:
    def __init__(self, owner, balance):
        self.owner = owner
        self.__balance = balance          # приватный атрибут
    
    def deposit(self, amount):
        if amount > 0:
            self.__balance += amount
    
    def withdraw(self, amount):
        if 0 < amount <= self.__balance:
            self.__balance -= amount
            return amount
        raise ValueError("Недостаточно средств")
    
    def get_balance(self):
        return self.__balance

acc = BankAccount("Иван", 1000)
acc.deposit(500)
print(acc.get_balance())   # 1500
# acc.__balance            # Ошибка – доступ закрыт
```

**Пример 2. Свойство с валидацией возраста**
```python
class Person:
    def __init__(self, name, age):
        self.name = name
        self._age = age         # защищённый атрибут
    
    @property
    def age(self):
        return self._age
    
    @age.setter
    def age(self, value):
        if not (0 <= value <= 150):
            raise ValueError("Некорректный возраст")
        self._age = value

p = Person("Мария", 25)
p.age = 30        # работает через сеттер
# p.age = -5      # ValueError
```

### Наследование (повторное использование)

**Пример 1. Геометрические фигуры**
```python
class Shape:
    def area(self):
        return 0

class Rectangle(Shape):
    def __init__(self, width, height):
        self.width = width
        self.height = height
    
    def area(self):
        return self.width * self.height

class Circle(Shape):
    def __init__(self, radius):
        self.radius = radius
    
    def area(self):
        return 3.14 * self.radius ** 2

rect = Rectangle(4, 5)
print(rect.area())   # 20
```

**Пример 2. Транспортные средства**
```python
class Vehicle:
    def __init__(self, brand, speed):
        self.brand = brand
        self.speed = speed
    
    def info(self):
        return f"{self.brand}, скорость {self.speed} км/ч"

class Car(Vehicle):
    def __init__(self, brand, speed, fuel):
        super().__init__(brand, speed)
        self.fuel = fuel
    
    def info(self):
        return f"{super().info()}, топливо: {self.fuel}"

class Bike(Vehicle):
    def ring(self):
        return "Дзынь-дзынь!"

car = Car("Toyota", 120, "бензин")
print(car.info())   # Toyota, скорость 120 км/ч, топливо: бензин
```

### Полиморфизм (один интерфейс – разная реализация)

**Пример 1. Утиная типизация – разные животные**
```python
class Dog:
    def speak(self):
        return "Гав"

class Cat:
    def speak(self):
        return "Мяу"

class Cow:
    def speak(self):
        return "Му"

def make_sound(animal):
    # не важно, кто пришёл, главное – есть метод speak()
    print(animal.speak())

make_sound(Dog())   # Гав
make_sound(Cat())   # Мяу
make_sound(Cow())   # Му
```

**Пример 2. Полиморфизм через наследование (документы)**
```python
class Document:
    def export(self):
        pass

class PDF(Document):
    def export(self):
        return "Экспорт в PDF"

class Word(Document):
    def export(self):
        return "Экспорт в Word"

class HTML(Document):
    def export(self):
        return "Экспорт в HTML"

def save_document(doc: Document):
    print(doc.export())

save_document(PDF())    # Экспорт в PDF
save_document(Word())   # Экспорт в Word
save_document(HTML())   # Экспорт в HTML
```
